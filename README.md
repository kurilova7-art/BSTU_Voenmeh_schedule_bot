## 🎓 Бот расписания БГТУ «ВОЕНМЕХ»

Telegram‑бот для студентов и преподавателей. Позволяет быстро узнать расписание **по группе, преподавателю или аудитории** с удобной интерактивной клавиатурой (переключение дней недели, чётность, недельный просмотр). Данные автоматически обновляются с официального сайта каждый час.



## Возможности

- **Поиск**  
  – по номеру группы (например, `09С51`)  
  – по фамилии преподавателя  
  – по номеру аудитории (например, `468*`)

- **Интерактивное расписание**  
  – кнопки переключения дней (◀️ Сегодня ▶️)  
  – выбор чётности недели (Нечётная / Чётная)  
  – просмотр **всей недели** с группировкой пар и выделением текущей недели

- **Сохранение состояния**  
  – после перезапуска бот помнит последний выбранный объект (группа/преподаватель/аудитория) и выбранный день/неделю

- **Автообновление**  
  – каждые 60 минут бот скачивает XML с сайта военмеха и обновляет базу данных

- **Часто задаваемые вопросы**  
  – команда `/faq` содержит ответы на часто задаваемые вопросы, а также расписание пар и полезные ссылки

- **Новости вуза**  
  – команда `/news` содержит последние 5 новстей с официального сайта Военмеха


## Установка и запуск (локально)

### 1. Клонируйте репозиторий
```
git clone https://github.com/your-repo/schedule_bot.git
cd schedule_bot
```
### 2. Создайте виртуальное окружение
```
python -m venv .venv
source .venv/bin/activate      # Linux/Mac
.venv\Scripts\activate          # Windows
```
### 3. Установите зависимости 
```
pip install aiogram, aiosqlite, aiohttp, aiofiles, python-dotenv
```
### 4. Настройте переменные окружения
Создайте файл `.env` в корневой папке:
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
```
### 5. Запустите бота
```
python main.py
```


## Развертывание на Railway (облачный сервер)

Чтобы бот работал 24/7, его можно бесплатно развернуть на платформе [Railway](https://railway.app/).

### 1. Подготовка репозитория
Убедитесь, что в корне проекта есть:
- `main.py` – точка входа
- `requirements.txt` – список зависимостей
- `Procfile` – файл с содержимым:  
```
worker: python main.py
```


### 2. Загрузите код на GitHub
Создайте репозиторий и запушьте код (не забудьте добавить `.env` в `.gitignore`).

### 3. Подключите репозиторий к Railway
- Зарегистрируйтесь на [Railway](https://railway.app/) (через GitHub).
- Нажмите New Project → Deploy from GitHub repo.
- Выберите ваш репозиторий и ветку (например, `main`).
- Railway автоматически обнаружит `Procfile` и запустит бота.

### 4. Добавьте переменные окружения
В панели управления Railway перейдите в **Variables** и добавьте:
```
BOT_TOKEN=токен
ADMIN_ID=id
```

### 5. Перезапустите деплой
После добавления переменных нажмите Redeploy (или Deploy), чтобы бот перезапустился с новыми настройками.

После успешного деплоя бот будет работать круглосуточно. 


## Команды бота

Все команды доступны через меню (кнопка **Меню** слева от поля ввода) или вводом `/` в чате.

| Команда | Описание |
|---------|-----------|
| `/search` | Поиск по группе, преподавателю или аудитории |
| `/group` | Поиск расписания по номеру группы |
| `/teacher` | Поиск расписания по фамилии преподавателя |
| `/classroom` | Поиск расписания по номеру аудитории |
| `/faq` | Часто задаваемые вопросы |
| `/news` | Последние новости вуза |
| `/help` | Показать список команд |


## Технические детали
- Язык: Python 3.14
- Библиотеки: aiogram 3.x, aiosqlite, aiohttp, python-dotenv
- База данных: SQLite (таблицы lessons, metadata)
- Источник данных: https://voenmeh.ru/obrazovanie/timetables/
- Частота обновления: 60 минут
